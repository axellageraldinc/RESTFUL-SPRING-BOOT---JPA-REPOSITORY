# RESTFUL-SPRING-BOOT---JPA-REPOSITORY

## Description
Project built with Spring Boot to create REST API.

## Technology Stack
* Spring Boot
* JPA Repository
* Lombok

## Prerequisites
* JDK installed
* add Lombok Dependency on pom.xml
* Install lombok plugin on INTELLIJ IDEA > Preferences> Plugins > Search for Lombok Plugin > Install > Restart INTELLIJ IDEA
